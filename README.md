# SKETCH Website

A multi-page website for the college club **SKETCH**, built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Scroll-Driven Hero**: Smooth image sequence animation on the landing page.
- **Glassmorphism Design**: Modern, dark-themed UI with subtle glass effects.
- **Aceternity UI Components**: Integration of advanced animations and interactive components.
- **Responsive Layout**: Fully optimized for mobile and desktop devices.
- **Animated Transitions**: Seamless page and component transitions using Framer Motion.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **Icons**: Tabler Icons
- **Components**: Aceternity UI

## 📦 Getting Started

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## 🌐 Deployment

The project is configured for **Vercel** with zero-config. Simply push the repository to GitHub and connect it to Vercel.

- **Root Directory**: `.` (Root)
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
